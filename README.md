**Game Overview**
Dobby’s Night Out is a role-playing-game (RPG) where you play and control the actions from the point of view of a hairless cat named Dobby. The main character, Dobby, is required to go through a series of quests, each varying in difficulty. You need to make decisions based on the goal of each level. The levels involve: Turkey (Level 1), Europe (Level 2), Ocean (Level 3), Maritimes (Level 4) and a final boss level, Toronto. 

**Game Mechanics**
In third person, Dobby can move left, right, forwards and backwards using ‘WASD’ or arrow keys. Additionally, the character can jump, crouch, and it will utilize natural physics, regular collision handling, and interaction points. There are additional cat powers that can be purchased using spiders and controlled using additional keys such as other letters. Going through each level, there will be cut scenes, interactive puzzles, narrative dialogue, item pickup, and hazards. XP levels, quest-based progression. Quests include riddles and puzzles.
